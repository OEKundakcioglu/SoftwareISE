# SoftwareISE — Supplementary Materials

This repository contains the supplementary materials for the paper:

**“Software Competencies Missing in Industrial and Systems Engineering Curricula”**  
Buket İpek Akbal and O. Erhun Kundakcioglu

The materials here support transparency and reproducibility by providing the full survey instrument (all items), demographic summaries, and high‑resolution figures cited in the manuscript.

---

## Repository contents (high level)

- **Survey instrument & aggregated results**
  - `Summary/Survey Monkey Summary.pdf` — SurveyMonkey export of the survey titled **“Impact of Programming and Database Proficiency on Professional Success”** (includes the complete set of survey items, response distributions, and text responses as provided by the platform export).

- **Demographic charts (examples)**
  - `Figures/Q3.png` — Undergraduate degree field/discipline distribution.
  - `Figures/Q4.png` — Highest degree and major distribution.
  - `Figures/Q10.png` — Self-reported programming knowledge level (Novice → Advanced).

- **Research-question figures (examples)**
  - `Figures/RQ1-12.png` — Response distribution across work domains (Yes / No / Uncertain-Does not apply).
  - `Figures/RQ2-7.png` — **Database education adequacy** by country (Yes / Partially / No).
  - `Figures/RQ2-8.png` — **Database education adequacy** by sector (Yes / Partially / No).
  - `Figures/RQ4-5.png` — **Perceived importance** by work domain (Very significant → Unsure).

> Note: File names follow the numbering used in the manuscript (e.g., `Q3` corresponds to survey item Q3 in the PDF export; `RQ…` files correspond to figures grouped under the paper’s research questions).

## How to cite

If you use or reference these materials, please cite the paper and (optionally) the repository. You can use a generic citation:

**Plain text**
> Akbal, B. İ. & Kundakcioglu, O. E., *“Software Competencies Missing in Industrial and Systems Engineering Curricula”*. GitHub repository: https://github.com/OEKundakcioglu/SoftwareISE

**BibTeX (template)**
```bibtex
@misc{AkbalKundakcioglu_SoftwareISE,
  author       = {Akbal, Buket Ipek and Kundakcioglu, O. Erhun},
  title        = {Software Competencies Missing in Industrial and Systems Engineering Curricula},
  url          = {https://github.com/OEKundakcioglu/SoftwareISE},
  note         = {Accessed: YYYY-MM-DD}
}
```

## Notes on data and privacy

- The provided results are **aggregated** outputs exported from SurveyMonkey.
- Any personally identifying information is not intended to be shared in this repository.  
  If you notice any sensitive content, please open an issue so it can be removed.
